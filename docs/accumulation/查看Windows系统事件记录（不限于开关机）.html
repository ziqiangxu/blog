# 查看 Windows 系统事件记录

如果你像我一样在测试一个程序的稳定性，但是早上醒来或者在一段你不在的时候电脑意外停止了工作。这时可以：

  • 使用命令行启动 eventvwr.msc 程序
  • 点击 Windows日志
  • 点击 系统

找到你需要查看的事件。根据事件记录的时间，你就可以做一个大致的判断，但是要千万小心地下决定，因为这个时间有可能是错误的。

# 题外话

我以为是我的古董电脑因为长时间运行而宕机了，然后通过查询事件发现很可能是因为自动更新失败而出现宕机。
