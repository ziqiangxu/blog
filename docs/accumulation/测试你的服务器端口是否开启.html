# 测试服务器端口是否开启

有时候开启一个服务却访问失败，老是不知道到底是端口开放问题还是程序问题。如要测试你的服务器端口是否可以在外网访问。你的只需要在你的本地 Linux 安装一个 netcat 软件

使用方法 netcat -zv <IP地址或主机名> <端口>

示例： netcat -zv yixiu.life 22

# Tips
