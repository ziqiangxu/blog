# 常见文件格式的官网

  • DICOM 官网： https://www.dicomstandard.orgopen in new window
  • NIfTI 官网： https://nifti.nimh.nih.govopen in new window

通常 DICOM 文件后缀为 dcm， NIfTI 文件后缀为 nii 或 nii.gz(压缩过的 nii 文件)。后文将 DICOM 简写为 dcm，NIfTI 简写为 nii
