# 栈和队列

# 栈（stack）

可定义为只能在末端插入和删除的线性表。也可根据存储的方式分为顺序栈（基于数组存储）和链式栈（基于链表存储）。

# 队列

只允许在表的一端插入，在另一端删除。也可根据存储的方式分为顺序队列（基于数组存储）和链式队列（基于链表存储）。
