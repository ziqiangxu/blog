              # 医疗图像软件

              注意，以下许多网站需要科学上网才能访问

              # 常见文件格式的官网

              • DICOM 官网： https://www.dicomstandard.org (opens new window)
              • NIfTI 官网： https://nifti.nimh.nih.gov (opens new window)

              通常 DICOM 文件后缀为 dcm， NIfTI 文件后缀为 nii 或 nii.gz(压缩过的 nii 文件)。后文将 DICOM 简写为 dcm，NIfTI 简写为 nii

              # 读写和格式转化

              名称 类型 描述 官网
              pydicom Python 库 读写 dcm https://pydicom.github.io (opens new window)
              nibabel Python 库 读写 nii https://nipy.org/nibabel (opens new window)
              dicom2nifti Python 库 dcm 转 nii https://github.com/icometrix/dicom2nifti (opens new window)
              dcm2niix 命令行工具 dcm 转 nii https://github.com/rordenlab/dcm2niix (opens new window)

              # 可视化

              常用的库： vtk vtkjs OpenCV matplotlib

              名称 类型 特点 官网
              DICOM Web Viewer 浏览器 轻量 https://ivmartel.github.io/dwv/ (opens new window)
              Paraview-Glance 浏览器 强大 https://kitware.github.io/paraview-glance/nightly (opens new window)
              itk-vtk-viewer 浏览器 https://kitware.github.io/itk-vtk-viewer/app (opens new window)
              ITK-SNAP 桌面应用 轻量 http://www.itksnap.org (opens new window)
              3D Slicer 桌面应用 强大 https://www.slicer.org (opens new window)

              https://medevel.com/14-best-browser-web-based-dicom-viewers-projects (opens new window)推荐了一些基于浏览器的dicom viewer，不过有些软件看起来好像已经很久没有维护了，此类软件不推荐使用

              # 组织机构

              • https://www.kitware.com/platforms (opens new window)
              • https://www.mitk.org (opens new window)
              • Open Health Imaging Foundation https://ohif.org (opens new window)， 相关demo (opens new window)

              # PACS

              Picture Archiving and Comunication System 的缩写

              • Orthanc 一个开源的 PACS 软件，官网 https://www.orthanc-server.com (opens new window)
              • https://medevel.com/10-open-source-pacs-dicom (opens new window) 搜集了一些开源 PACS

              # 其他

              • https://idoimaging.com (opens new window) 收录了许多医疗图像相关的软件
              • NVIDIA 公司推出的模型训练平台Clara： https://docs.nvidia.com/clara/tlt-mi/clara-train-sdk-v3.0/aiaa/key_features.html (opens new window)，可以方便地用来训练分割模型，提高医学图像的标注效率
              • 放射学相关： https://www.radiomics.io (opens new window)
