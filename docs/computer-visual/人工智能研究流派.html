        # 人工智能研究流派

        目前，人工智能研究主要有三种途径：

        # 符号主义（思维理论）

        符号主义认为人类认知的基本元素是符号，认知的过程就是符号处理的过程（一阶谓词逻辑）。

        # 连接主义（阈值理论）

        连接主义认为人类认知的基本元素是神经元本身。人类的认知过程就是大量的神经元的整体活动。（研究方法：人工神经网络）

        # 行为主义（进化理论）

        该理论认为人的本质能力是在动态环境中的行走能力、对外界事物的感知能力、维持生命和繁衍生息的能力，正是这些能力对智能的发展提供了基础，因此智能是某种复杂系统所浮现的性质。
